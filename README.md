# 🦊 Foxy CLI Pro v2.5
أداة Dashboard تفاعلية مخصصة لنظام Kali Linux لفتح أدوات الاختراق والتطوير بسرعة.

## 🚀 المميزات
- قائمة مرقمة لاختيار البرامج (Chrome, Burp Suite, ChatGPT, إلخ).
- واجهة ASCII Art احترافية (ثعلب).
- لا تخرج من الأداة إلا عند كتابة `0` أو `exit`.

## 🛠 التثبيت (Installation)
```bash
git clone [https://github.com/ansm8482-lab/foxy.git](https://github.com/ansm8482-lab/foxy.git)
cd foxy
chmod +x foxy
sudo cp foxy /usr/local/bin/foxy
